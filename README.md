# Spring Boot

Build Restful CRUD API for a simple Note-Taking application using Spring Boot, JPA and Hibernate.

## Requirements

1. Java - 1.8.x

2. Maven - 3.x.x

## Explore Rest APIs

The app defines following CRUD APIs.

    GET /notes
    
    POST /notes
    
    GET /notes/{Id}
    
    PUT /notes/{Id}
    
    DELETE /notes/{Id}

You can test them using postman or any other rest client.
